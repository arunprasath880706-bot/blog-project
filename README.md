# Blog Project - Complete Web Application

## Project Overview
A fully functional blog application with user authentication, CRUD operations, search, pagination, and role-based access control.

## Features
- ✅ User Registration & Login with password hashing
- ✅ Create, Read, Update, Delete blog posts
- ✅ Search posts by title or content
- ✅ Pagination for posts listing
- ✅ Role-based access control (Admin/Editor/User)
- ✅ Prepared statements for SQL injection prevention
- ✅ Server-side and client-side form validation
- ✅ Responsive design with Bootstrap
- ✅ Cross-browser compatible

## Technologies Used
- PHP
- MySQL
- HTML5
- CSS3 / Bootstrap
- JavaScript
- Git & GitHub

## How to Run
1. Start XAMPP (Apache & MySQL)
2. Import the database from `database.sql`
3. Update `config.php` with your database credentials
4. Open `http://localhost/blog-project/`
